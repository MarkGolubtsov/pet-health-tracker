# Pet Health Tracker Backend Seeds

Каталог содержит seed-данные для локальной разработки.
Seed-данные не являются production-данными.

## Правила

1. Seed-данные должны быть синтетическими.
2. Seed-данные не должны содержать реальные медицинские документы.
3. Seed-данные не должны содержать реальные персональные данные.
4. Seed-данные должны быть применимы после миграций.
